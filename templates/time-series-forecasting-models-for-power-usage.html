Multi-Step LSTM time series forecasting models for power usage¶

Given the rise of smart electricity meters and the wide adoption of electricity generation technology like solar panels, there is a wealth of electricity usage data available.

This data represents a multivariate time series of power-related variables that in turn could be used to model and even forecast future electricity consumption.

Unlike other machine learning algorithms, long short-term memory recurrent neural networks are capable of automatically learning features from sequence data, support multiple-variate data, and can output a variable length sequences that can be used for multi-step forecasting.

In this tutorial, you will discover how to develop long short-term memory recurrent neural networks for multi-step time series forecasting of household power consumption.

Tutorial Overview¶

This tutorial is divided into nine parts; they are:

  1. Problem description
  2. Load and prepare dataset
  3. Vanilla LSTMs
  4. Encoder-Decoder LSTM model
  5. CNN-LSTM Encoder-Decoder model
  6. ConvLSTM Encoder-Decoder model

1. Problem description¶

The Household Power Consumption dataset is a multivariate time series dataset that describes the electricity consumption for a single household over four years.

The data was collected between December 2006 and November 2010 and observations of power consumption within the household were collected every minute. It is a multivariate series comprised of seven variables (besides the date and time); they are:

  • global_active_power: The total active power consumed by the household (kilowatts).
  • global_reactive_power: The total reactive power consumed by the household (kilowatts).
  • voltage: Average voltage (volts).
  • global_intensity: Average current intensity (amps).
  • sub_metering_1: Active energy for kitchen (watt-hours of active energy).
  • sub_metering_2: Active energy for laundry (watt-hours of active energy).
  • sub_metering_3: Active energy for climate control systems (watt-hours of active energy).

Active and reactive energy refer to the technical details of alternative current.

A fourth sub-metering variable can be created by subtracting the sum of three defined sub-metering variables from the total active energy as follows:

sub_metering_remainder = (global_active_power * 1000 / 60) - (sub_metering_1 + sub_metering_2 + sub_metering_3)

  zip_path = keras.utils.get_file(origin=uri, fname="household_power_consumption.zip")
  zip_file = ZipFile(zip_path)
  zip_file.extractall()
  csv_path = "household_power_consumption.txt"
  
  # load all data
  dataset = pd.read_csv(csv_path,  
                        sep=';', 
                        header=0, 
                        low_memory=False, 
                        infer_datetime_format=True, 
                        parse_dates={'datetime':[0,1]}, 
                        index_col=['datetime'])
  
  # mark all missing values
  dataset.replace('?', np.nan, inplace=True)
  # make dataset numeric
  dataset = dataset.astype('float32')
  
  # fill missing values with a value at the same time one day ago
  def fill_missing(values):
      one_day = 60 * 24
      for row in range(values.shape[0]):
          for col in range(values.shape[1]):
              if np.isnan(values[row, col]):
                  values[row, col] = values[row - one_day, col]
  
  # fill missing
  fill_missing(dataset.values)
  
  # add a column for for the remainder of sub metering
  values = dataset.values
  dataset['sub_metering_4'] = (values[:,0] * 1000 / 60) - (values[:,4] + values[:,5] + values[:,6])

  class VanillaLSTM():
      """
      Multivariate Multi-Step VanillaLSTM docstrings 
      """
      def __init__(self):
          pass
      
      # resample data to daily
      def dataresample(self, data, groupedtype):
          self.data = data
          self.groupedtype = groupedtype
          daily_groups = data.resample(groupedtype)
          self.daily_data = daily_groups.sum()
          return self.daily_data
      
      # split dataset into train/val/test sets
      def datasplit(self):
          train = self.daily_data.loc["2006-12-17":"2009-01-03"]
          val = self.daily_data.loc["2009-01-04":"2010-01-02"]
          test = self.daily_data.loc["2010-01-03":"2010-11-20"]
          
          #train = self.daily_data.loc["2006-12-17":"2007-01-01"]
          #val = self.daily_data.loc["2007-01-02":"2007-01-09"]
          #test = self.daily_data.loc["2007-01-10":"2007-01-16"]
          
          # convert dataframes to data arrays 
          self.train = np.array(train)
          self.val = np.array(val)
          self.test = np.array(test)
          
          return self.train, self.val, self.test
          
      def dataresize(self, n_input, n_output, target, predictors, **kwargs):
          self.n_input = n_input 
          self.n_output = n_output                
          self.target = {"index": target, "name": [self.data.columns[x] for x in target]}
          self.predictors = {"indices": predictors, "names": [self.data.columns[x] for x in predictors]}
          self.scanning_pitch = kwargs.get('scanning_pitch', 1)
                          
          self.train_x, self.train_y = self.__class__.rezise(self.train, self.n_input, self.n_output, 
                                                             self.scanning_pitch, self.target["index"], 
                                                             self.predictors["indices"])
          self.val_x, self.val_y = self.__class__.rezise(self.val, self.n_input, self.n_output, 
                                                         self.scanning_pitch, self.target["index"], 
                                                         self.predictors["indices"])
          self.test_x, self.test_y = self.__class__.rezise(self.test, self.n_input, self.n_output, 
                                                           self.scanning_pitch, self.target["index"], 
                                                           self.predictors["indices"])
  
      # build and train the model
      def modelbuild(self, epochs, batch_size, loss, metrics, **kwargs):
          self.epochs = epochs
          self.batch_size = batch_size
          self.loss = loss
          self.metrics = metrics
          self.patience = kwargs.get('patience', 5)
          self.verbose = kwargs.get('verbose', 0)
          
          # define model
          self.model = Sequential()
          self.model.add(LSTM(200, activation='relu', input_shape=(self.train_x.shape[1], self.train_x.shape[2])))
          self.model.add(Dense(100, activation='relu'))
          self.model.add(Dense(self.train_y.shape[1]))
          self.model.compile(loss=loss, optimizer='adam', metrics=metrics)
          # simple early stopping
          early_stopping = EarlyStopping(monitor='val_loss', mode='min', patience=self.patience, verbose=1)
          # fit network
          self.history = self.model.fit(self.train_x, self.train_y, validation_data=(self.val_x, self.val_y), 
                                        epochs=self.epochs, batch_size=self.batch_size, verbose=self.verbose, 
                                        callbacks=[early_stopping])
          return self.model
  
      def evaluate(self):
          #data = np.concatenate((self.val[-self.n_input:,:], self.test))
          data_x, data_y = self.__class__.rezise(data=self.test, n_input=self.n_input, n_output=self.n_output, 
                                                 scanning_pitch=self.n_output, target=self.target["index"], 
                                                 predictors=self.predictors["indices"])
              
          predictions = self.model.predict(data_x, verbose=0)
          self.pred = {"Actual values": data_y, "Predicted values": predictions}
          # calculate overall RMSE
          self.rmse_score = mean_squared_error(data_y, predictions, squared=False)
          # calculate an RMSE score for each day
          self.rmse_scores = mean_squared_error(data_y, predictions, multioutput='raw_values', squared=False)
              
          return self.rmse_score, self.rmse_scores
  
      # plot the losses

          if len(self.model.metrics_names) <= 2:
              l, c = 1, 2
          else:
              l, c = (len(self.model.metrics_names)//2)+1, 2

                  plt.plot(epochs, self.history.history["val_"+self.model.metrics_names[i]], "green", label="Validation "+"val_"+self.loss[0]+'*')
                  plt.xlabel("Epochs")
                  plt.ylabel(self.loss[0]+'*')

                  plt.plot(epochs, self.history.history["val_"+self.model.metrics_names[i]], "green", label="Validation "+"val_"+self.model.metrics_names[i])
                  plt.xlabel("Epochs")
                  plt.ylabel(self.model.metrics_names[i])

          plt.suptitle("Training and Validation losses: {}".format(title))

      @staticmethod

          x = list()
          y = list()
          start, end = n_input, data.shape[0] - n_output + 1
          for i in range(start, end, scanning_pitch):  
              sample_x = data[i-n_input:i, predictors]
              sample_y = data[i:i+n_output, target]
              sample_y = sample_y.reshape((len(sample_y)))
              x.append(sample_x)
              y.append(sample_y)
          # Tranform the data set features and labels into arrays
          x = np.array(x)
          y = np.array(y)
          return x, y

from keras.layers import RepeatVector
  from keras.layers import TimeDistributed
  
  class EncoderDecoderLSTM(VanillaLSTM):
         
      @staticmethod
      def rezise(data, n_input, n_output, scanning_pitch, target, predictors):     
          x = list()
          y = list()
          start, end = n_input, data.shape[0] - n_output + 1
          for i in range(start, end, scanning_pitch):  
              sample_x = data[i-n_input:i, predictors]
              sample_y = data[i:i+n_output, target]
              #sample_y = sample_y.reshape((len(sample_y)))
              x.append(sample_x)
              y.append(sample_y)
          # Tranform the data set features and labels into arrays
          x = np.array(x)
          y = np.array(y)
          return x, y    
      
      # build and train the model
      def modelbuild(self, epochs, batch_size, loss, metrics, **kwargs):
          self.epochs = epochs
          self.batch_size = batch_size
          self.loss = loss
          self.metrics = metrics
          self.patience = kwargs.get('patience', 5)
          self.verbose = kwargs.get('verbose', 0)
  
          # define model
          self.model = Sequential()
          self.model.add(LSTM(200, activation='relu', input_shape=(self.train_x.shape[1], self.train_x.shape[2])))
          self.model.add(RepeatVector(self.train_y.shape[1]))
          self.model.add(LSTM(200, activation='relu', return_sequences=True))
          self.model.add(TimeDistributed(Dense(100, activation='relu')))
          self.model.add(TimeDistributed(Dense(1)))
          self.model.compile(loss=loss, optimizer='adam', metrics=metrics)
          # simple early stopping
          early_stopping = EarlyStopping(monitor='val_loss', mode='min', patience=self.patience, verbose=1)
          # fit network
          self.history = self.model.fit(self.train_x, self.train_y, validation_data=(self.val_x, self.val_y), 
                                        epochs=self.epochs, batch_size=self.batch_size, verbose=self.verbose, 
                                        callbacks=[early_stopping])
          return self.model
      
      def evaluate(self):
          #data = np.concatenate((self.val[-self.n_input:,:], self.test))
          data_x, data_y = self.__class__.rezise(data=self.test, n_input=self.n_input, n_output=self.n_output, 
                                                 scanning_pitch=self.n_output, target=self.target["index"],

from keras.layers import ConvLSTM2D
      
  class ConvLSTMEncoderDecoder(VanillaLSTM):
         
      @staticmethod
      def rezise(data, n_input, n_output, scanning_pitch, target, predictors):     
          x = list()
          y = list()
          start, end = n_input, data.shape[0] - n_output + 1
          for i in range(start, end, scanning_pitch):  
              sample_x = data[i-n_input:i, predictors]
              sample_y = data[i:i+n_output, target]
              #sample_y = sample_y.reshape((len(sample_y)))
              x.append(sample_x)
              y.append(sample_y)
          # Tranform the data set features and labels into arrays
          x = np.array(x)
          x = x.reshape(x.shape[0], int(n_input/n_output), 1, n_output, x.shape[2])
          y = np.array(y)
          return x, y    
      
      # build and train the model
      def modelbuild(self, epochs, batch_size, loss, metrics, **kwargs):
          self.epochs = epochs
          self.batch_size = batch_size
          self.loss = loss
          self.metrics = metrics
          self.patience = kwargs.get('patience', 5)
          self.verbose = kwargs.get('verbose', 0)
  
          # define model
          input_shape = (self.train_x.shape[1], self.train_x.shape[2], self.train_x.shape[3], self.train_x.shape[4])
          self.model = Sequential()
          self.model.add(ConvLSTM2D(filters=64, kernel_size=(1,3), activation='relu', input_shape=input_shape))
          self.model.add(Flatten())
          self.model.add(RepeatVector(self.train_y.shape[1]))
          self.model.add(LSTM(200, activation='relu', return_sequences=True))
          self.model.add(TimeDistributed(Dense(100, activation='relu')))
          self.model.add(TimeDistributed(Dense(1)))
          self.model.compile(loss=loss, optimizer='adam', metrics=metrics)
          # simple early stopping
          early_stopping = EarlyStopping(monitor='val_loss', mode='min', patience=self.patience, verbose=1)
          # fit network
          self.history = self.model.fit(self.train_x, self.train_y, validation_data=(self.val_x, self.val_y), 
                                        epochs=self.epochs, batch_size=self.batch_size, verbose=self.verbose, 
                                        callbacks=[early_stopping])
          return self.model
      
      def evaluate(self):
          #data = np.concatenate((self.val[-self.n_input:,:], self.test))
          data_x, data_y = self.__class__.rezise(data=self.test, n_input=self.n_input, n_output=self.n_output, 
                                                 scanning_pitch=self.n_output, target=self.target["index"], 
                                                 predictors=self.predictors["indices"])
  
          data_y = data_y[:,:,0]
          predictions = self.model.predict(data_x, verbose=0)
          predictions = predictions[:,:,0]
          self.pred = {"Actual values": data_y, "Predicted values": predictions}
          # calculate overall RMSE
          self.rmse_score = mean_squared_error(data_y, predictions, squared=False)
          # calculate an RMSE score for each day
          self.rmse_scores = mean_squared_error(data_y, predictions, multioutput='raw_values', squared=False)
  
          return self.rmse_score, self.rmse_scores
